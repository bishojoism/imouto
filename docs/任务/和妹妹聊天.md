# 任务之和妹妹聊天

## 安全阈值

|            | 建议   | 上限   |
| ---------- | ------ | ------ |
| 每次不超过 | 五分钟 | 十分钟 |
| 每天不超过 | 六次   | 十二次 |

## 操作步骤

1. 查看微信和小红书私信列表，明确答应当你妹妹、三天之内回过消息、符合[安全聊天对象标准](#安全聊天对象标准)，这三点全部满足才需要聊，否则可以先试试针对问题沟通能不能解决，还是不行就不要聊了。严重违反安全聊天对象标准的，连沟通都不要沟通，直接删。
2. 平时[发消息](/规范/发消息)。起床早安、睡觉晚安，多叫妹妹、多说喜欢。你只需要主动做这些最基础的事，其他的事情不用，她希望你做你再做。并且即使有她的希望，你也不能泄露隐私、不能违反法律、不能造成伤害、不能违反平台规定、不能逾越安全阈值、不能违背你的意愿。如果你实在想找话题，可以分享日常、分享作品，要是还是聊不上就算了。
3. 特殊日子可以[寄手写贺卡](/规范/寄手写贺卡)。任何时候都不可以赠送除手写贺卡以外的其他任何物品，包括虚拟物品。
4. 尽可能在小红书上聊，但如果对方要求加微信那就加，加完要问清楚对方习惯用小红书还是用微信。
5. 重复上述步骤，直到解决私信列表。

## 安全聊天对象标准

此处列举的是常见风险，而非全部风险。对于未明确列举的情况，应根据其实际影响，按照财产安全、内容安全、情绪安全、认知安全、关系安全等原则综合判断。只要具有与已列举行为相同或相近的风险，应视为不符合安全聊天对象标准。对于无法确定是否安全的情况，应采取更保守的处理方式，暂缓执行，直到能够确认不存在风险。

### 财产安全

1. 无任何借钱、提现、带货、推荐投资、刷单或变相索要礼物、红包、打赏的行为。
2. 不以任何理由暗示或请求你提供经济援助或其他财物，或体现她需要。
3. 不诱导你转账、提供账号密码、下载未知软件、扫描未知二维码、点击不明链接。

### 内容安全

1. 交流绝不触及政治敏感、色情低俗、违法犯罪、赌博暴力等话题。
2. 遵守平台规定，不诱导你做出导致平台限制或封禁账号功能的行为。
3. 不要求你传播未经证实的信息，也不要求你泄露他人隐私、侵犯他人名誉。

### 情绪安全

1. 整体来说对你有礼貌、态度好，不长期给你带来压力、恐惧、焦虑、负担。
2. 不存在持续的严重负能量宣泄，不讨论自残自杀，也不引你产生自残自杀念头。
3. 不通过示弱、自责、威胁、冷暴力等方式制造愧疚感，从而操控你的行为或决定。

### 认知安全

1. 接受你的时间安排与聊天频率，允许你回复速度慢、暂停聊天、长期不联系。
2. 尊重你的人格，不贬低你的外表或品质、不否定你的行为或价值观。
3. 尊重你的拒绝，不会因为你拒绝某件事而持续劝说、纠缠或报复。

### 关系安全

1. 不要求你只和自己聊天，不要求你删除其他好友，不挑拨你与家人朋友的关系。
2. 不鼓励你放弃学习、工作、睡眠、健康或现实中的正常社交来陪伴聊天。
3. 发生冲突时能和和气气地沟通，而非辱骂、羞辱、威胁、恐吓、断联、拉黑。
