# 任务之和妹妹聊天

## 安全阈值

|            | 建议   | 上限   |
| ---------- | ------ | ------ |
| 每次不超过 | 五分钟 | 十分钟 |
| 每天不超过 | 六次   | 十二次 |

## 操作步骤

1. [发消息](/规范/发消息)和答应过当你妹妹的、三天内给你发过消息的女生聊天。起床早安、睡觉晚安，多叫妹妹、多说喜欢。你只需要主动做这些最基础的事，其他的事情不用，她希望你做你再做，并且不能泄露隐私、不能违反法律、不能造成伤害、不能违反平台规定、不能逾越安全阈值、不能违背你的意愿。实在想找话题，可以分享日常、分享作品，要是还是聊不上就不要聊了。
2. 在特殊日子，可以[寄手写贺卡](/规范/寄手写贺卡)，不可以送其他物品（包括虚拟物品），不可以有金钱往来。
3. 尽可能在小红书上聊，但如果对方要求加微信那就加，加完要问清楚对方习惯用小红书还是用微信。
4. 重复上述步骤，直到解决私信列表。
