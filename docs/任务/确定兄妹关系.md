在打完招呼之后，一般来说对方问个年龄该答应的也就答应了，事多的就算了吧，没意思。虽然再磨一磨说不定也会答应，但是事多的就算答应了，在往后的相处当中也更有可能闹各种不愉快，不如放弃，反正打招呼是批量打的，少她一个不少。此处涉及到的聊天非常基础，不是那种需要先熟悉慢慢试探培养感觉的，也不是那种需要讨论很多细节的，她要么爽快答应要么没戏。就是有些女生的注意力不在确定兄妹关系这件事上，你要负责把她的注意力拉回来。

1. 查看小红书私信列表，之前打的招呼有人回，就[发消息](#规范之发消息)和她协商，紧扣确定兄妹关系的主题。
2. 有些时候她问完你情况只回个语气词，此时你可以提醒她一句「那你要当我妹妹吗」。
3. 在她答应之前，时刻谨记保持应有的生分，因为她还不是妹妹。
4. 在五次来回之内，如果她仍然没有答应，就直接放弃。
5. 重复上述步骤，直到处理完所有回招呼的女生。
