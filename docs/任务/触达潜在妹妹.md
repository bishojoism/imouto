# 任务之触达潜在妹妹

传统思维可能会自己精心拍照片、写文案、发作品、做账号，苦苦等待女生来互动，这种从零开始做流量的方式费时费力，我们不妨心思活络一点。社交媒体每天提供一定的招呼次数，可以用来给尚未互关的用户推送消息提醒，而在别人已经火起来的收妹妹作品评论区，往往聚集了大量留言表达要哥哥意向的女生。趁还没被别人抢走抢走，去私信她们，哪怕上来第一句直接问她你能不能当我妹妹，也有可能有人会答应。小红书平台限制每天可以打十次招呼，一旦用满，几乎可确保当天答应的人数不小于一，运气好一天爆出两三个都有可能。也不要给自己压力，就算一个都没收到，也没什么后果，收多了反而还聊不过来。

## 安全阈值

|            | 建议标准 | 硬性上限 |
| ---------- | -------- | -------- |
| 每次不超过 | 十分钟   | 二十分钟 |
| 每天不超过 | 两次     | 四次     |

## 操作步骤

1. 小红书搜「收妹妹处兄妹」。
2. 翻热门作品评论区，找一天内评论要哥哥的女生（为了有效利用招呼机会，可以稍微看一眼她的主页，判断她当妹妹的意愿）。
3. 点进主页关注并私信「我想收一些妹妹，你能当我妹妹吗」。
4. 重复上述步骤，直到用完小红书每日十次招呼机会。
