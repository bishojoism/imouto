```

## [准备](准备脚本)脚本

*准备脚本包含绕不开人工的步骤。*

快手.准备 = () => {
    手机.应用市场安装('快手')
    手机.注册并登录('快手')
}
```

## [收妹妹](#收妹妹脚本)脚本

> 请先[准备](#准备脚本)

*收妹妹脚本的所有步骤均可纯自动化，但需要视觉模型。*

::: tip
在热门作品评论区私信撬客，批量精准触达新鲜需求。
:::

快手搜「收妹妹处兄妹」，翻作品评论区，按从新到旧排，找要哥哥的，点进主页关注并私信「我想收一些妹妹，你能当我妹妹吗」，重复直到招呼上限。

::: danger
尽管招呼次数每天都会刷新，但每周打招呼的天数不得超过三天，否则每多一天掉五点SAN值。
:::

1. [进入主页](../操作员/应用#进入主页能力)`快手应用`
2. [点击](../操作员/界面#点击能力)「右上角搜索图标」
3. [输入](../操作员/应用#输入能力)「收妹妹处兄妹」
4. [点击](../操作员/界面#点击能力)「右上角搜索按钮」
5. [点击](../操作员/界面#点击能力)「第一条作品」
6. [点击](../操作员/界面#点击能力)「右侧评论区图标」
7. [点击](../操作员/界面#点击能力)「评论条数标签」
8. [点击](../操作员/界面#点击能力)「按最新排序按钮」
9. [检查为真则点击](../操作员/界面#检查为真则点击能力)「有评论要哥哥的」「用户头像」，为假则跳至第19步
10. [检查](../操作员/界面#检查能力)「未关注」，为假则跳至第18步
11. [点击](../操作员/界面#点击能力)「关注按钮」
12. [点击](../操作员/界面#点击能力)「发私信按钮」
13. [点击](../操作员/界面#点击能力)「底部消息输入框」
14. [输入](../操作员/应用#输入能力)「我想收一些妹妹，你能当我妹妹吗」
15. [点击](../操作员/界面#点击能力)「右下角发送图标」
16. [看见](../操作员/应用#看见能力)「上限」，为真则[优雅结束整个流程](../操作员/应用#优雅结束整个流程能力)`快手应用`
17. [回到上一页](../操作员/应用#回到上一页能力)
18. [回到上一页](../操作员/应用#回到上一页能力)
19. [检查](../操作员/界面#检查能力)「翻到底了或没有一天内的评论了」，为真则两次[回到上一页](../操作员/应用#回到上一页能力)并跳至第4步，为假则[向下滚动](../操作员/应用#向下滚动能力)并跳至第9步

```js
快手.收妹妹 = () => {
    第1步:
    应用.进入主页(快手应用)
    第2步:
    界面.点击('右上角搜索图标')
    第3步:
    应用.输入('收妹妹处兄妹')
    第4步:
    for (; ;) {
        界面.点击('右上角搜索按钮')
        第5步:
        界面.点击('第一条作品')
        第6步:
        界面.点击('右侧评论区图标')
        第7步:
        界面.点击('评论条数标签')
        第8步:
        界面.点击('按最新排序按钮')
        第9步:
        for (; ;) {
            if (界面.检查为真则点击('有评论要哥哥的', '用户头像')) {
                第10步:
                if (界面.检查('未关注')) {
                    第11步:
                    界面.点击('关注按钮')
                    第12步:
                    界面.点击('发私信按钮')
                    第13步:
                    界面.点击('底部消息输入框')
                    第14步:
                    应用.输入('我想收一些妹妹，你能当我妹妹吗')
                    第15步:
                    界面.点击('右下角发送图标')
                    第16步:
                    if (应用.看见('上限')) 应用.优雅结束整个流程(快手应用)
                    第17步:
                    应用.回到上一页()
                }
                第18步:
                应用.回到上一页()
            }
            第19步:
            if (界面.检查('翻到底了或没有一天内的评论了')) { 应用.回到上一页(); 应用.回到上一页(); continue 第4步 } else { 应用.向下滚动(); continue 第9步 }
        }
    }
}
```
