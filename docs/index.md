# 小红书收妹妹攻略

比起处对象，处兄妹除了叫宝宝改为叫妹妹，亲密没有区别，而承诺就可以避免以下要素，爽：

1. 博弈行为：拉扯/摩擦
2. 义务行为：排他性/仪式感

## 思路

1. 社交媒体每天提供一定的招呼次数，可以用来给尚未互关的用户推送一条消息提醒。
2. 而在别人已经火起来的收妹妹作品评论区，聚集了大量留言表达蹲哥哥意向的用户。
3. 此时去私信她，哪怕上来第一句直接问她确定兄妹关系，也有可能会答应。
4. 小红书每天有十次招呼次数，一旦用满，几乎可确保当天答应的人数不小于一。

## 流程

1. 小红书搜“收妹妹处兄妹”，点进热度高的作品，将评论从新到旧排序。
2. 找出蹲哥哥的评论，卡的不要，慢热的不要，要自介的不要，照片钓鱼的不要。
3. 关注，发送刚刚浏览的评论，私信“我想收一些妹妹，你能当我妹妹吗”。
4. 重复上述步骤，直至用满招呼次数（小红书每天最多能打十次招呼）。

## 聊天

1. 有想说的内容：提问/回答
2. 不知道说什么：语气词/表情包
