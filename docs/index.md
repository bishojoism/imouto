# 网上处兄妹标准作业程序

本程序提供了一些任务，用于作为网上处兄妹的参考（涉及的APP仅包含微信和小红书）。这些任务哪天心情好了再做，不做也没事，累了停下来歇一歇，不想做了随时可以中断，如果成为了负担，请立即停止。每项任务都包含了标题、描述、安全阈值、操作步骤，有些还会额外包含禁忌事项，安全阈值与禁忌事项必须严格遵守。在做任务之前，请按照建议定好闹钟。一旦超过建议，就要拉起警报；倘若打破上限，不要犹豫，直接拉总闸，当月好好休息吧。

## 主线任务（合计约一小时）

这些只要做一次就行了，首周可以额外一天做。

1. [准备微信](/任务/准备微信)
2. [准备小红书](/任务/准备小红书)

这些每天都可以做，但是每周不得超过四天做。
