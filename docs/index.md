# 网上处兄妹：一种安全、轻量、简洁、高效的异性相处范式

“让关系为人生服务。”

## 痛点分析

资产阶级话语统摄着异性相处，表面上装着真情，暗地里净是腐朽，机会被一小撮有钱有颜高学历高情商的人垄断，大多数人处于被考核的地位，只有一退再退，才能维持对方的喜欢。

## 解决思路

更换旗号，专找愿意放下身段抛弃爱情崇高、跳过暧昧流程快速建立关系、不加精挑细选轻易说出喜欢的异性，双方持续情绪上的对等义务劳动，从而淘汰以喜不喜欢论相不相处的人。

## 适用范围

仅作为个人层面的临时过渡，不能替代对结构的反抗，想要真正改变现实，组织才是唯一武器。另外，对方并不是天生就该听你的，工业化的流程之外，还需要真实地满足对方需求。

## 操作步骤

1. （一次性）准备微信，可白板号可老号，但最好将朋友圈设为三个月可见。
2. （一次性）准备小红书，可白板号可老号，但最好将你关注了谁、谁关注了你设为仅自己可见。
3. （每批5分钟，每天2批）小红书搜“收妹妹处兄妹”作品，对于评论区里新留言要哥哥的女生：关注并私信“我想收一些妹妹，你能当我妹妹吗”，直到用完小红书每天的10次招呼机会。
4. （每批5分钟，每天2批）小红书盯着私信列表，对于应了招呼的女生：不断把她的注意力拉回到确定兄妹关系这件事上。
5. （每批5分钟，每天8批）微信小红书盯着私信列表，对于说过愿意、回了消息、守着纪律的女生：起床早安、睡觉晚安，多叫妹妹、多说喜欢，有想说的提问回答、没想说的语气词表情包，分享有意思的日常、分享有意思的作品。
