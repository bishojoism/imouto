# 网上处兄妹标准作业程序

本程序提供了一些任务，用于作为网上处兄妹的参考（涉及的APP仅包含微信和小红书）。这些任务哪天心情好了再做，不做也没事，累了停下来歇一歇，不想做了随时可以中断，如果成为了负担，请立即停止。每项任务不光有操作步骤，还有必须严格遵守的安全阈值，每次定好闹钟到时间停，每天列好表格到次数停。一旦超过建议标准，就要拉起警报；倘若打破硬性上限，不要犹豫，直接拉总闸，当月好好休息吧。

## 主线任务（一小时）

这些只要做一次就行了，首周可以额外安排一天。

1. [准备微信](/任务/准备微信)（三十分钟）
2. [准备小红书](/任务/准备小红书)（三十分钟）
