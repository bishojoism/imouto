# 网上处兄妹：一种安全、轻量、简洁、高效的异性相处范式

“关系服务人生。”

## 痛点分析

现实中异性交往门槛较高，往往需要长期考核、反复试探，机会多向条件更突出的人倾斜，普通人在这个过程中处于被动地位，维持对方的喜欢需要不断地退让。

## 解决思路

换一种名义，专找愿意跳过暧昧、快速建立关系、不做过多筛选、轻易表达好感的异性，双方持续**对等**地投入情绪劳动，淘汰只按喜不喜欢来决定相不相处的人。

## 适用范围

仅作为个人层面的一种临时过渡，没法真正改变现实环境，同时要注意，对方不是自动就该配合你的设定，除了走流程之外，也要真实地发掘并满足对方的需求。

## 操作步骤

1. （一次性）准备微信，可白板号可老号，但最好将朋友圈设为三个月可见。
2. （一次性）准备小红书，可白板号可老号，但最好将你关注了谁、谁关注了你设为仅自己可见。
3. （每批5分钟，每天2批，每周4天）小红书搜“收妹妹处兄妹”作品，对于评论区里新留言要哥哥的女生：关注并私信“我想收一些妹妹，你能当我妹妹吗”，直到用完小红书每天的10次招呼机会。注意，不可省略前半句“我想收**一些**妹妹”。
4. （每批5分钟，每天2批，每周4天）小红书盯着私信列表，对于应了招呼的女生：不断把她的注意力拉回到确定兄妹关系这件事上。
5. （每批5分钟，每天8批，每周4天）微信小红书盯着私信列表，对于说过愿意、回了消息、守着**纪律**的女生：起床早安、睡觉晚安，多叫妹妹、多说喜欢，有想说的提问回答、没想说的语气词表情包，分享有意思的日常、分享有意思的作品。尽量在小红书聊，如果对方要求加微信那就加，加完问清楚之后用小红书聊还是用微信聊。特殊日子可以寄手写贺卡，除此之外不可以赠送任何物品，包括虚拟物品，也不可以转账。

## 劳动纪律

- 遵守法律、遵守平台规定。
- 哥哥无条件爱妹妹、妹妹无条件偏袒哥哥。
