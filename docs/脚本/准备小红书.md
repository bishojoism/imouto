# 准备小红书

需要一个小红书账号，白板号即可，主页不需要发内容，老号也行，发了东西也不用删，但最好不要让别人看到你关注了谁和谁关注了你。

## 安全阈值

|            | 建议   | 上限     |
| ---------- | ------ | -------- |
| 每次不超过 | 十分钟 | 十五分钟 |
| 每天不超过 | 一次   | 两次     |
| 每周不超过 | 一天   | 两天     |

## 操作步骤

1. 没安装就安装，在手机自带的应用市场就有。
2. 没注册就注册，昵称头像简介填低龄化的，性别年龄地区填真的。
3. 没登录就登录，手机号一键登录即可。
4. 设置里关闭关注的人和粉丝列表显示。
