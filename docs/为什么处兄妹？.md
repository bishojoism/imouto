# 为什么处兄妹？

建立新的关系、与人保持足够的亲密、有归属感、得到喜欢与爱、得到尊重、受欢迎且被人认可、实现人生理想、成为有能力的人、给自己一个目标、开心生活、寻求有意思的事情、尝试新鲜事物、勇于冒险、充满激情、尊重自己、坚定自己走的人生之路、贯彻自己的理念、自主做决定、有安全感、认定很多人与事出发点是好的、带给他人以幸福、为他人奉献自己、积极面对挑战、获得成长、成为有操守的人、勇于接纳自己……那么多价值观，与异性相处总能符合那么一两个，因此毋庸置疑有意义。

第一时间想到处对象，但要我说，处对象并不是好主意。自由恋爱赋予了一个人决定自己喜不喜欢另一个人的自由，也就是说，你必须一直投其所好、对症下药，对方喜欢什么你就怎样，如此维持对方的喜欢，像永无止境的绩效考核，必须一直合格。而如今的恋爱市场，低交换价值的人往往处于被考核的地位，要么你选择坚守价值观，结果一个也谈不到，要么你为了能谈到对象一退再退，退到最后完全违背了自己的价值观，无论怎样都没有意义。而所谓的提升自己，更是在迎合资产阶级价值观，仿佛有钱有颜高学历高情商的人才配与异性相处。被恋爱对象考核与被恋爱市场考核，哪个不在被考核？

既然处对象这条路注定走向异化，我们不得不探索全新的替代方案。它的门槛要足够低，低到连金字塔底的大多数人都进得去；它的门槛还要足够高，高到连金字塔尖的一小撮人都进不去。既然认真的人是因为喜欢，那么替代方案就要要求对方进行情绪上的义务劳动，从而将认真的人拒之门外，专找愿意放下身段抛弃爱情崇高、跳过暧昧流程快速建立关系、不加精挑细选轻易说出喜欢的人，跟过家家似的。

社交媒体上刚好存在这样的过家家文化，那就是处兄妹。某些用户会发收妹妹的作品，另一些用户会在作品下方发要哥哥的评论，人还不少，我们可以去凑个热闹。但我们始终不能忘记我们的价值观，有些人可能盲目追求数量，只要对方说过愿意、回了消息，就把对方算作妹妹，我们还要对方守着纪律。另一方面，我们也不能仗着理论的优越欺负人，妹妹并不是天生就该听你话的，你也要发掘并满足妹妹被看见、被理解、被陪伴、被认可的情感需求。
