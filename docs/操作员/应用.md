# 应用操作员

::: tip
应用操作员的所有能力均可纯自动化，且无需视觉模型。
:::

```js
const 应用 = {}
```

[[TOC]]

## [进入主页](#进入主页能力)能力

进入主页（包名）

- 字符串参数`包名`——表示要进入主页的是哪个包。
- 没有返回。

> 关闭`包名`并启动`包名`并等待一秒。

```js
应用.进入主页 = (包名) => {
    shizuku.cmd(['shell', 'am', 'force-stop', 包名])
    shizuku.cmd(['shell', 'monkey', '-p', 包名, '-c', 'android.intent.category.LAUNCHER', '1'])
    global.sleep(1000)
}
```

## [输入](#输入能力)能力

输入（文本）

- 字符串参数`文本`表示要输入的文本。
- 没有返回。

> 复制粘贴`文本`并等待一秒。

```js
应用.输入 = (文本) => {
    global.setClip(文本)
    shizuku.cmd(['shell', 'input', 'keyevent', 'KEYCODE_PASTE'])
    global.sleep(1000)
}
```

## [优雅结束整个流程](#优雅结束整个流程能力)能力

优雅结束整个流程（包名）

- 字符串参数`包名`表示要关闭的包。
- 没有返回。

> 关闭`包名`并退出程序。

```js
应用.优雅结束整个流程 = (包名) => {
    shizuku.cmd(['shell', 'am', 'force-stop', 包名])
    global.exit()
}
```

## [回到上一页](#回到上一页能力)能力

回到上一页（）

- 没有返回。

> 按返回键并等待一秒。

```js
应用.回到上一页 = () => {
    shizuku.cmd(['shell', 'input', 'keyevent', 'KEYCODE_BACK'])
    global.sleep(1000)
}
```

## [向下滚动](#向下滚动能力)能力

向下滚动（）

- 没有返回。

> 在一秒内从屏幕 [width \* 0.5, height \* 0.6] 滑到 [width \* 0.5, height \* 0.6] 并等待一秒。

```js
应用.向下滚动 = () => {
    shizuku.cmd(['shell', 'input', 'swipe' , device.width * 0.5, device.height * 0.6, device.width * 0.5, device.height * 0.4, 1000])
    global.sleep(1000)
}
```

## [看见](#看见能力)能力

看见（文本）

- 字符串参数`文本`——表示要判断是否存在的文本
- 布尔返回——真表示存在；假表示不存在。

> 看屏幕上`文本`是存在还是不存在返回真也就是存在假也就是不存在

```js
应用.看见 = (文本) => {
    const path = files.path("截图.png")
    shizuku.cmd(['shell', 'screencap', '-p', path])
    const texts = ocr.recognizeText(path)
    for (let i = 0; i < texts.length; i++) {
        if (texts[i].indexOf(文本) !== -1) return true
    }
    return false
}
```
