# 界面操作员

*界面操作员的所有能力均可纯自动化，但需要视觉模型。*

::: info
调用视觉模型需要用户填写「API_KEY」、「BASE_URL」、「MODEL」，存储于「视觉模型」桶内。劣等模型即可，不需要旗舰模型，但端侧模型倒是不行。
:::

```js
const 界面 = {}

const 视觉模型 = storages.create('视觉模型')
let API_KEY = 视觉模型.get('API_KEY') || ''
let BASE_URL = 视觉模型.get('BASE_URL') || ''
let MODEL = 视觉模型.get('MODEL') || ''

let flag = true
const w = floaty.window(
    <vertical bg="white" padding="24">
        <input id="API_KEY" hint="API_KEY" text={API_KEY}/>
        <input id="BASE_URL" hint="BASE_URL" text={BASE_URL}/>
        <input id="MODEL" hint="MODEL" text={MODEL}/>
        <horizontal>
            <button id="确定">确定</button>
            <button id="取消">取消</button>
        </horizontal>
    </vertical>
)
w.requestFocus()
w.确定.click(() => {
    API_KEY = w.API_KEY.getText()
    BASE_URL = w.BASE_URL.getText()
    MODEL = w.MODEL.getText()
    视觉模型.put('API_KEY', API_KEY)
    视觉模型.put('BASE_URL', BASE_URL)
    视觉模型.put('MODEL', MODEL)
    flag = false
})
w.取消.click(() => {
    throw new Error('用户取消')
})
while (flag) global.sleep(100)
```

## [检查](#检查能力)能力

检查（内容）

- 字符串参数`内容`——表示要检查的是什么内容。
- 布尔返回——真表示正确；假表示错误。

> 根据屏幕检查`内容`是正确还是错误并返回真也就是正确或假也就是错误。

```js
界面.检查 = (内容) => {
    const path = files.path('截图.png')
    shizuku.cmd(['shell', 'screencap', '-p', path])
    let err
    for (let i = 0; i < 5; i++) {
        try {
            const response = http.postJson(`${BASE_URL}/chat/completions`, {
                model: MODEL,
                messages: [
                    {
                        role: 'system',
                        content: '你是界面操作员。'
                    },
                    {
                        role: 'user',
                        content: {
                            type: 'image_url',
                            image_url: {
                                url: `data:image/jpeg;base64,${images.toBase64(images.resize(images.read(path), [1000, 1000]), 'jpeg', 95)}`
                            }
                        }
                    },
                    {
                        role: 'user',
                        content: `根据屏幕检查${内容}是正确还是错误并返回真也就是正确或假也就是错误。`
                    }
                ],
                response_format: {
                    type: 'json_schema',
                    json_schema: {
                        name: 'bool',
                        strict: true,
                        schema: {
                            type: 'boolean',
                            description: '结果',
                        }
                    }
                }
            }, {
                headers: {
                    'Authorization': `Bearer ${API_KEY}`
                }
            })
            return JSON.parse(response.body.json().choices[0].message.content)
        } catch (e) {
            err = e
        }
    }
    throw err
}
```

## [点击](#点击能力)能力

点击（元素）

- 字符串参数`元素`——表示要点击的是界面上哪个元素。
- 没有返回。

> 定位屏幕上的`元素`并点击并等待一秒。

```js
界面.点击 = (元素) => {
    const path = files.path('截图.png')
    shizuku.cmd(['shell', 'screencap', '-p', path])
    let err
    for (let i = 0; i < 5; i++) {
        try {
            const response = http.postJson(`${BASE_URL}/chat/completions`, {
                model: MODEL,
                messages: [
                    {
                        role: 'system',
                        content: '你是界面操作员。'
                    },
                    {
                        role: 'user',
                        content: {
                            type: 'image_url',
                            image_url: {
                                url: `data:image/jpeg;base64,${images.toBase64(images.resize(images.read(path), [1000, 1000]), 'jpeg', 95)}`
                            }
                        }
                    },
                    {
                        role: 'user',
                        content: `定位屏幕上的${元素}`
                    }
                ],
                response_format: {
                    type: 'json_schema',
                    json_schema: {
                        name: 'point',
                        strict: true,
                        schema: {
                            type: 'array',
                            description: '坐标[x, y]',
                            items: {
                                type: 'number',
                                minimum: 0,
                                maximum: 1000,
                            },
                            minItems: 2,
                            maxItems: 2,
                        }
                    }
                }
            }, {
                headers: {
                    'Authorization': `Bearer ${API_KEY}`
                }
            })
            const [x, y] = JSON.parse(response.body.json().choices[0].message.content)
            shizuku.cmd(['shell', 'input', 'tap', Math.floor(x * device.width / 1000), Math.floor(y * device.height / 1000)])
            global.sleep(1000)
            return
        } catch (e) {
            err = e
        }
    }
    throw err
}
```

## [检查为真则点击](#检查为真则点击能力)能力

检查为真则点击（内容，元素）

- 字符串参数`内容`——表示要检查的是什么内容。
- 字符串参数`元素`——表示要点击的是界面上哪个元素。
- 布尔返回——真表示正确；假表示错误。

> 根据屏幕检查`内容`是正确还是错误如果是正确则定位屏幕上的`元素`并点击并等待一秒并返回真也就是正确否则返回假也就是错误。

```js
界面.检查为真则点击 = (内容, 元素) => {
    const path = files.path('截图.png')
    shizuku.cmd(['shell', 'screencap', '-p', path])
    let err
    for (let i = 0; i < 5; i++) {
        try {
            const response = http.postJson(`${BASE_URL}/chat/completions`, {
                model: MODEL,
                messages: [
                    {
                        role: 'system',
                        content: '你是界面操作员。'
                    },
                    {
                        role: 'user',
                        content: {
                            type: 'image_url',
                            image_url: {
                                url: `data:image/jpeg;base64,${images.toBase64(images.resize(images.read(path), [1000, 1000]), 'jpeg', 95)}`
                            }
                        }
                    },
                    {
                        role: 'user',
                        content: `根据屏幕检查${内容}是正确还是错误，若正确则定位屏幕上的${元素}，若错误则输出null。`
                    }
                ],
                response_format: {
                    type: 'json_schema',
                    json_schema: {
                        name: 'point',
                        strict: true,
                        schema: {
                            anyOf: [
                                {
                                    type: 'array',
                                    description: '坐标[x, y]',
                                    items: {
                                        type: 'number',
                                        minimum: 0,
                                        maximum: 1000,
                                    },
                                    minItems: 2,
                                    maxItems: 2,
                                },
                                {
                                    type: 'null',
                                    description: '错误'
                                },
                            ],
                        }
                    }
                }
            }, {
                headers: {
                    'Authorization': `Bearer ${API_KEY}`
                }
            })
            const point = JSON.parse(response.body.json().choices[0].message.content)
            if (!point) return false
            const [x, y] = point
            shizuku.cmd(['shell', 'input', 'tap', Math.floor(x * device.width / 1000), Math.floor(y * device.height / 1000)])
            global.sleep(1000)
            return true
        } catch (e) {
            err = e
        }
    }
    throw err
}
```
