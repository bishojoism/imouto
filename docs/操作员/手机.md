## [安装注册登录](#安装注册登录能力)能力

安装注册登录（应用）

- 字符串参数`应用`——表示要安装、注册、登录的是哪个应用。
- 没有返回。

> 弹出浮动按钮，请人类安装、注册、登录`应用`，并在点击按钮后继续程序。

```js
手机.安装注册登录 = (应用) => {
    const w = floaty.window(
        <frame>
            <button id="继续">{`⁝ 请安装、注册、登录「${应用}」后点此继续。`}</button>
        </frame>
    )
    let flag = true, x = 0, y = 0, wx, wy, time
    w.继续.setOnTouchListener((_, event) => {
        switch (event.getAction()) {
            case event.ACTION_DOWN: {
                x = event.getRawX()
                y = event.getRawY()
                wx = w.getX()
                wy = w.getY()
                time = new Date().getTime()
                return true
            }
            case event.ACTION_MOVE: {
                w.setPosition(wx + event.getRawX() - x, wy + event.getRawY() - y)
                return true
            }
            case event.ACTION_UP: {
                if (Math.abs(event.getRawX() - x) < 5 && Math.abs(event.getRawY() - y) < 5) {
                    flag = false
                }
                return true
            }
        }
    })
    while (flag) global.sleep(100)
}
```
