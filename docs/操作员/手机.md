# 手机操作员

*手机操作员的所有能力均绕不开人工。*

```js
const 手机 = {}
```

[[TOC]]

## [应用市场安装](#应用市场安装能力)能力

应用市场安装（应用）

- 字符串参数`应用`——表示要安装的是哪个应用。
- 没有返回。

> 若手机上没有`应用`则用手机自带的应用商店下载并等待安装完成

```js
declare function 应用市场安装(应用)
手机.应用市场安装 = 应用市场安装
```

## [注册并登录](#注册并登录能力)能力

注册并登录（应用）

- 字符串参数`应用`——表示要注册并登录的是哪个应用。
- 没有返回。

> 若没有注册`应用`则注册，若没有登录`应用`则登录

```js
declare function 注册并登录(应用)
手机.注册并登录 = 注册并登录
```
